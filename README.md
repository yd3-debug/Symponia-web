# Symponia Marketing System

Automated content pipeline for TikTok and Instagram.
Reads Symponia's philosophy and animal archetypes directly from the app source.
Content is generated by Claude using the same voice as the app.

## Structure

```
knowledge/          ← brand voice, app description (source of truth)
prompts/            ← Claude prompt templates per content type
scripts/            ← content generation scripts (run locally)
workflows/          ← n8n workflow JSONs (import into n8n)
templates/          ← visual templates (Canva / HTML)
output/             ← generated content (gitignored)
```

## Setup

### 1. Install dependencies
npm install

### 2. Configure environment
cp .env.example .env
# Fill in your API keys

### 3. Generate content locally
npm run daily                          # daily philosophical quote
npm run animal -- --animal wolf        # wolf archetype post
npm run shadow -- --animal snake       # snake shadow post

### 4. Set up n8n
- Import workflows/daily-content-pipeline.json
- Import workflows/instagram-post-approved.json
- Add your API keys as n8n credentials

### 5. Set up Airtable
Create a base called "Symponia Content" with table "Content Queue":
- Quote (text)
- Caption (long text)
- Hashtags (text)
- Visual Prompt (long text)
- Image URL (URL) ← you add this after creating the visual
- Status (single select): draft / approved / posted
- Type (text)
- Generated At (date)
- Posted At (date)

### 6. Set up Instagram
- Create Meta Business Account
- Connect Instagram Business account
- Get long-lived access token from Meta Graph API Explorer
- Add META_ACCESS_TOKEN and META_INSTAGRAM_ACCOUNT_ID to .env

## Content Flow

1. n8n runs daily at 9am → calls Claude → saves draft to Airtable
2. You review in Airtable, create the visual (Canva), add Image URL
3. Change status to "approved"
4. n8n detects approval → posts to Instagram automatically
5. For TikTok: use the generated script, record/edit the video, post manually or via Buffer
